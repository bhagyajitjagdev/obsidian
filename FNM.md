# Installing fnm
```bash
curl -fsSL https://fnm.vercel.app/install | bash
```

# Verify that everything works
```bash
fnm --help
```

# Install the `lts` version
```bash
fnm install --lts
```

# Listing all Node.js versions
```bash
fnm ls-remote
```

# Using a particular version of Node.js
```bash
fnm use <version>
```
